# Vedixa Technologies Website

Complete multi-page React + Vite + Tailwind CSS website recreated from the approved page designs.

## Pages
- Home
- About Us
- Services
- Technologies
- Architecture
- Portfolio
- Industries
- Careers
- Contact Us

## Run
```bash
npm install
npm run dev
```

## Production build
```bash
npm run build
npm run preview
```

All page content is implemented as editable React components. The generated visual references are included in `public/assets`, and cropped visual artwork is used in the hero sections.
